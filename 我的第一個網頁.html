CSS-負責外觀、樣式

層疊樣式表（英文:Casading Style Sheets，簡寫CSS），又稱串樣式列表、級聯樣式表、階層式樣式表，一種用來為結構化文件（如HTML 文件或 XML 應用）添加樣式（字型、間距和顏色等）的電腦語言，由 W3C定義和維護。目前最新版本是 CSS2.1，為 W3C 的推薦標準。CSS3 現在已被大部分現代瀏覽器支援，而下一版的 CSS4 仍在開發中。

JavaScript-負責行為、互動

JavaScript，通常縮寫為JS，是一種進階直譯語言的程式語言。JavaScript 是一門基於原型、函式先行的語言，是一門多範式的語言，它支援物件導向編程，指令式程式設計，以及函式語言程式設計。

jQuery-JavaScript函式庫

jQuery 是一套跨瀏覽器的 JavaScript 函式庫，簡化 HTML 與 JavaScript 之間的操作。全球前10,000個存取最高的網站中，有65%使用了jQuery，是目前最受歡迎的 JavaScript 函式庫。

Bootstrap-CSS UI 框架

Bootstrap 是一組用於網站和網路應用程式開發的開元前端（所謂「前端」，指的是展現給終端使用者的介面。與之對應的「後端」是在伺服器上面執行的程式碼）框架，包括 HTML、CSS及JavaScript 的框架，提供字體排印、表單、按鈕、導航及其他各種元件及Javascript 擴充套件、旨在使動態網頁和Web應用的開發更加容易。
